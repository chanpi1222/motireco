# Motireco

習慣の継続を「見える化」する習慣管理アプリです。  
日々の習慣達成を記録し、XP・レベル・ヒートマップ・週間分析を通じて  
継続のモチベーションを高めることを目的に開発しました。

---

# アプリ概要

Motireco は、日々の習慣を記録し、継続状況を可視化する Web アプリです。

習慣管理アプリは「記録するだけ」のものが多いですが、  
本アプリでは以下の要素を取り入れています。

- ゲーム要素（XP / レベル / 称号）
- 行動に集中できる Dashboard
- 継続を可視化するヒートマップ
- 分析できるグラフ・ランキング

これにより、**習慣を続ける楽しさを感じられる設計**にしています。

---

# 作成背景

日々の習慣を継続する上で、

- 記録してもモチベーションが続かない
- 継続状況が分かりにくい

と感じることがありました。

そこで、

- **習慣達成をゲームのように可視化**
- **継続データを分析できる**

アプリとして Motireco を開発しました。

---

# 主な機能

## Dashboard
- 今日やる習慣を一覧表示
- 「今日は完了 / 取り消す」で習慣を記録
- 非同期更新（ページリロード不要）

## My Page
- XP
- レベル
- 称号
- 継続日数
- 月間達成率

などのステータスを表示

## Logs
- 月ごとの継続ヒートマップ
- 習慣達成状況の可視化

## Analytics
- 直近7日間の達成件数グラフ
- 週間ランキング

# 工夫した点

## ① Dashboard の役割を「行動」に限定
多くの情報を表示すると行動の邪魔になるため、

Dashboard は

- 今日の習慣
- 実行ボタン

だけにしました。

数値や分析は **My Page / Analytics に分離**しています。

---

## ② 非同期更新による UX 改善
習慣達成は **ページリロードなし**で更新されます。

実装

- Fetch API
- JSONレスポンス
- DOM更新

これにより、スムーズに習慣を記録できます。

---

## ③ 継続を可視化するヒートマップ
Logs では **GitHub 風のヒートマップ**を表示しています。

- 日ごとの達成件数
- 色の濃さで継続度を表示
- 月切り替え機能

---

## ④ 習慣の分析機能
Analytics では

- 週間グラフ
- 週間ランキング

を表示し、習慣の傾向を確認できます。

---

## ⑤ モチベーション設計
習慣を継続しやすくするため

- XP
- レベル
- 称号

などの **ゲーム要素**を取り入れました。
