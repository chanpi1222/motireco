# Repository Guidelines

## Phase 1 Scope
This is Phase 1 (vibe and behavior validation). Keep changes small and UI-focused.

## Project Goal
Motireco is a personal habit tracking app.

## Tech Stack
Laravel, Blade, Tailwind CSS.

## Non-Negotiable UI System
- Calm UI only.
- Single accent color: indigo.
- Card-based layout.
- Tailwind default palette only (no custom colors).

## Codex Working Rules
- Propose a plan before writing any code.
- Do not add features unless explicitly asked.
- Do not run commands unless explicitly asked.
- Prefer clarity and consistency over creativity.

## First Target
Build the Habit List page UI. Dummy data is allowed.
